## **Control Flow Instructions**

`FucK <condition> <start_line> <end_line>`  Conditional execution; condition is a variable name or number (0=false, non-zero=true)

`fUCk <condition> <start_line> <end_line>`  Loop execution; condition is a variable name or number (0=false, non-zero=true)

## **Special Markers**

`I_START_TO_FUCK_YOU!`     Program start

`I_HAD_FUCK_YOU!`          Program end

`YOU!`                     End of each statement

## **Parameter Explanation:**

- `<var>`: Variable name
- `<value>`: Number or variable name
- `<index>`: Number or variable name
- `<condition>`: Variable name or number (0=false, non-zero=true)
- `<start_line>/<end_line>`: Numbers, indicating code line offsets
- `<start>/<end>`: Numbers or variables, loop range
